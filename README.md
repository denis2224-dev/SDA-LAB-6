# Lab 6 - Queue ADT + FILE + UDT (C)

This project implements Lab 6 in procedural C using dynamic memory, linked-list-based queue ADTs, FILE operations, and a warehouse registry UDT reused from Lab 5.

## Project structure

```
LAB6/
  CMakeLists.txt
  README.md
  main.c
  warehouse.h
  warehouse.c
  queue.h
  queue.c
  file_ops.h
  file_ops.c
  data/
    experiment.txt
    output.txt
```

Implementation is added in incremental milestones with validation and Git commits.
